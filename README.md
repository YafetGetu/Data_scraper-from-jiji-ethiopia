A professional web scraping tool for extracting product listings from the Jiji Ethiopia online marketplace.  
This project is actively developed with regular improvements to strengthen my portfolio and maintain a consistent GitHub contribution streak.

---

## Features

- Product listing scraping (titles, prices, categories)
- Export data to JSON and CSV formats
- Robust error handling with retry logic
- Configurable request delays and timeouts
- Logging system for debugging and monitoring
- Designed for scalability and future expansion
